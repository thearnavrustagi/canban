## Tips

- Use a terminal with a dark background for best contrast
- Resize to ~120x35 for a good aspect ratio
- Use a tool like [vhs](https://github.com/charmbracelet/vhs) to record terminal GIFs
- For static screenshots, use your OS screenshot tool or `iTerm2 > Shell > Capture Output`
- Save as PNG, keep width around 800–1200px for sharp rendering on GitHub
- Consider adding a `demo.gif` in `assets/` for the hero section of the README
